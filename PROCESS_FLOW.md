This is a document outlining the process flow for the Enoverlabs project.

## Process Flow
- Install libraries ✅
- Create routes ✅
- Create default CSS styles, typography, colors
- Create components
- Build views/pages (landing/home, signup/login, pricing, study plan, take a test)

## Libraries
- React router
- Axios
- Framer motion

## Components
- Buttons
- Navbar
- Signup
- Login
- Image + Text
- Card (Icon + Text, Price, Subject)
- Avatar
- Leaderboard
- Footer

## Pages
- Landing/Home
- Signup/Login
- Pricing
- StudyPlan
- TakeATest
